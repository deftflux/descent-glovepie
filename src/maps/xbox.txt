
KEYBOARD MAPPINGS

XBOX Button  Key          Suggested Assignment
----------------------------------------------
D-Pad Up     Up arrow     Fire flare
D-Pad Down   Down arrow   Rear view
D-Pad Left   Left arrow   Cycle primary
D-Pad Right  Right arrow  Cycle secondary
Start+Y      Tab key      Automap
A            Enter key    Drop bomb
X            Space bar    Toggle bomb (Menu check boxes)
Start        Escape key   (Abort game / back out of menu)


JOYSTICK BUTTON MAPPINGS

XBOX Button  Joystick  Suggested Assignment
-------------------------------------------
RS           Button 0  Fire primary    
RT           Button 1  Fire secondary
Left Thumb   Button 2  Bank left
Right Thumb  Button 3  Bank right
B            Button 4  Convert energy to shield (D2)
Y            Button 5  Headlight (D2)
LS+LT        Button 6  Afterburner (D2)


JOYSTICK AXIS MAPPINGS

XBOX Axis      Joystick  Suggested Assignment
---------------------------------------------
Left Stick X   Axis 0    Slide L/R
Left Stick Y   Axis 1    Slide U/D
Right Stick X  Axis 2    Turn L/R
Right Stick Y  Axis 3    Pitch U/D
LS / LT        Axis 4    Throttle
